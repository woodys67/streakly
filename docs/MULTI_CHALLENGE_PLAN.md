# 멀티(팀) 챌린지 — 기획 문서

> 현재 앱은 **싱글 챌린지 모드**만 지원합니다.  
> 멀티 챌린지는 **유료 구독(PRO)** 모델로 추후 출시 예정입니다.

---

## 개요

| 항목 | 내용 |
|---|---|
| 기능명 | 멀티(팀) 챌린지 |
| 수익 모델 | 유료 구독 (PRO) |
| 현재 상태 | UI 플레이스홀더만 존재 (비활성) |

---

## 핵심 아이디어

혼자 하는 21일 챌린지를 **팀으로 확장** — 친구·동료와 같은 챌린지를 함께 시작하고,
서로의 진행 상황을 실시간으로 확인하며 동기부여를 주고받는다.

---

## 계획 기능 (미정, 추후 구체화)

### 팀 구성
- 팀 생성 / 초대 링크 공유
- 최대 인원 (예: 최대 10명, 추후 결정)
- 팀 이름 / 팀 아이콘 설정

### 팀 챌린지
- 팀원 전원이 동일한 챌린지를 동시 진행
- 팀원별 진행률 실시간 확인
- 팀 전체 완료율 집계

### 경쟁 & 동기부여
- 팀 내 랭킹 / 리더보드
- 응원 반응 (좋아요, 응원 메시지)
- 팀 전체 스트릭 달성 시 특별 배지

### 알림
- 팀원 체크인 알림
- 팀원이 뒤처질 때 독려 알림

- PRO: 멀티 챌린지 + 추가 기능 (금액 미정)

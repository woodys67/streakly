# Streakly 배지 시스템 기획안

> 작성일: 2026-06-01  
> 총 배지 수: 50개

---

## 설계 원칙

1. **앱 핵심 루프 강화** — 스트릭 유지, 챌린지 완주, 기록 습관을 배지로 직접 보상
2. **점진적 달성** — 같은 주제의 배지를 3~4단계로 구성해 지속 동기부여
3. **희귀도 4등급** — Common → Rare → Epic → Legendary (획득 난이도 차등)
4. **비밀 배지(Secret)** — 조건을 숨겨 놓아 우연한 발견의 재미 추가
5. **팀 챌린지 배지 분리** — 소셜 확장 시 새로운 보상 카테고리 제공

---

## 희귀도 정의

| 등급 | 배경 컬러 | 테두리 | 획득 난이도 |
|---|---|---|---|
| ⬜ **Common** | 밝은 회색 `#E8E8E8` | 없음 | 누구나 초반에 쉽게 |
| 🟦 **Rare** | 하늘색 `#DBEAFE` | 파란 `#4A90E2` | 꾸준히 하면 달성 |
| 🟧 **Epic** | 연오렌지 `#FFF3E0` | 오렌지 `#FF8C00` | 의지력 필요 |
| 🟨 **Legendary** | 금색 그라데이션 | 금 `#FFD700` | 극소수만 달성 |
| ⬛ **Secret** | 어두운 회색 (조건 미공개) | 점선 | 발견의 재미 |

## 배지 UI 반영 제안

### 획득 화면 (배지 잠금 해제 시)
```
┌──────────────────────────────────┐
│                                  │
│          🎉 배지 획득!            │
│                                  │
│         ┌──────────┐             │
│         │          │             │
│         │    🔥    │  ← 골드 테두리 애니메이션
│         │          │             │
│         └──────────┘             │
│                                  │
│       화톳불 / Bonfire           │
│   🟦 Rare                        │
│                                  │
│   스트릭 21일 연속 달성 완료!    │
│                                  │
│   ┌────────────┐ ┌────────────┐  │
│   │   공유하기  │ │   확인     │  │
│   └────────────┘ └────────────┘  │
└──────────────────────────────────┘
```

### 배지 컬렉션 화면 (기록 탭 내 또는 별도 탭)
- 획득한 배지: 풀 컬러 + 획득 날짜 표시
- 미획득 배지: 흑백 + 잠금 아이콘 (조건 힌트 제공)
- Secret 배지: 완전히 블러 처리 + "???" 표시

### 프로필 표시
- 가장 최근 획득 배지 3개를 프로필(기록 탭)에 노출
- 유저가 "대표 배지" 1개 선택 → 팀 챌린지 화면에서 이름 옆에 표시

## 1. 사용법

### 개요
Streakly는 매일 반복하는 루틴을 **챌린지** 단위로 관리하고, 연속 달성(스트릭)을 통해 습관을 형성하는 앱입니다.

### 기본 흐름

```
챌린지 생성 → 매일 루틴 체크 → 스트릭 누적 → 의지력·레벨 성장 → 배지 획득
```

### 화면 구성

| 탭 | 역할 |
|---|---|
| 🏠 홈 | 오늘의 루틴 목록, 스트릭 카드, 주간 달성률 확인 |
| 🚩 챌린지 | 진행 중인 챌린지 관리 및 달력 확인 |
| 📊 기록 | 완료한 챌린지, 배지 컬렉션, 통계 확인 |
| ⚙️ 설정 | 언어·알림·계정 설정 |

### 오늘의 루틴 완료하기

1. 홈 화면에서 오늘 해야 할 루틴 카드를 확인합니다.
2. 카드 좌측의 **원형 버튼**을 눌러 완료 처리합니다.
3. 서브루틴이 있는 경우, 각 서브루틴을 개별 체크해야 전체 완료로 처리됩니다.
4. 완료된 카드는 초록색 테두리와 취소선으로 표시됩니다.

### 새 챌린지 시작하기

홈 화면 우측 하단의 **＋ 버튼**을 눌러 새 챌린지를 만들 수 있습니다.

## 2. 챌린지 만들기

### 챌린지란?
특정 기간 동안 매일 수행할 루틴을 정의한 단위입니다. 챌린지를 만들면 시작일부터 목표 일수까지 매일 체크인하게 됩니다.

### 설정 항목

#### 챌린지 이름
수행할 루틴을 간결하게 입력합니다. (예: 독서 30분, 플랭크, 명상)

#### 목표 일수
| 옵션 | 일수 |
|---|---|
| 7일 | 단기 습관 시도 |
| 14일 | 습관 형성 중간 단계 |
| 21일 | 습관 형성 기본 단위 (Maxwell Maltz 이론) |
| 직접 입력 | 28일 이상 장기 챌린지 |

> 💡 연구에 따르면 새로운 습관이 완전히 자리 잡기까지 평균 **66일**이 필요합니다.

#### 서브루틴
하나의 챌린지를 여러 세부 항목으로 구성할 수 있습니다.  
예: "아침 루틴" 챌린지 → 스트레칭 10분 / 물 한 잔 / 일기 쓰기

- 서브루틴이 있으면 **모든 서브루틴을 완료해야** 해당 날짜가 달성됩니다.
- 각 서브루틴에 **알림 시간**을 개별 설정할 수 있습니다.

#### 반복 요일
매일 반복 또는 특정 요일(월~일)만 선택할 수 있습니다.  
요일을 지정하면 해당 요일에만 홈 화면에 루틴 카드가 표시됩니다.

#### 알림 시간
루틴 수행 전 알림을 받을 시간을 설정합니다.  
설정하지 않으면 알림이 발송되지 않습니다.

#### 노트
챌린지에 대한 목표나 다짐을 자유롭게 입력합니다.

### 챌린지 시작일
챌린지는 **만든 날 당일**부터 Day 1로 시작됩니다. 과거 날짜로 소급 설정은 불가합니다.

## 3. 스트릭 시스템

### 스트릭이란?
루틴을 **연속으로 완료한 일수**입니다. 하루도 빠짐없이 이어가는 것이 핵심입니다.

### 스트릭 계산 방식

```
오늘(Day N)부터 역순으로 연속 완료된 날수를 계산
```

| 상황 | 스트릭 |
|---|---|
| Day 5, 4, 3 모두 완료 (오늘 Day 5) | 3 |
| Day 5 미완료, Day 4, 3 완료 | 0 (오늘 빠짐) |
| Day 5 완료, Day 4 미완료 | 1 (오늘만) |

### 스트릭 초기화 조건
단 **하루라도 루틴을 완료하지 않으면** 스트릭이 0으로 초기화됩니다.  
이것이 Streakly의 핵심 긴장감입니다.

### 총 스트릭 (기록 화면)
기록 화면의 **총 스트릭**은 진행 중인 모든 챌린지의 현재 스트릭을 합산한 값입니다.

- 챌린지 A 스트릭 5 + 챌린지 B 스트릭 3 = 총 스트릭 **8**
- 누적 역사값이 아닌 **현재 상태의 합산**이므로, 스트릭이 끊기면 감소합니다.
- 의지력(Willpower)이 절대 감소하지 않는 것과 대조되는 **실시간 모멘텀 지표**입니다.

### 스트릭 마일스톤
스트릭이 특정 일수에 도달하면 배지가 지급됩니다.

| 스트릭 | 배지 |
|---|---|
| 1일 | 🔥 첫 불꽃 |
| 3일 | 🔥 불씨 |
| 7일 | 🔥 작은 불꽃 |
| 14일 | 🪵 모닥불 |
| 21일 | ✨ 화톳불 |
| 30일 | ⚡ 용광로 |
| 50일 | 🌋 불의 지배자 |
| 66일 | ☀️ 태양 (습관 완전 정착) |
| 100일 | 🌟 영원한 불꽃 |

## 4. 스트릭 복구

### 복구란?
어제 루틴을 빠뜨렸을 때, **의지력을 소모해 스트릭을 되살리는** 기능입니다.  
"어차피 끊겼으니 포기"라는 심리를 방지하고 지속성을 유지하는 데 도움을 줍니다.

### 복구 조건

| 조건 | 내용 |
|---|---|
| 복구 가능 시점 | 오늘 루틴을 아직 완료하지 않은 상태 |
| 복구 대상 날짜 | 어제 (currentDay - 1) |
| 필요 의지력 | 3 |
| 쿨다운 | 챌린지당 **7일에 1회** 제한 |

### 복구 방법

1. 홈 화면에서 오늘 미완료 상태인 루틴 카드를 확인합니다.
2. 카드 **우측 하단**에 `🔥 의지력 3 소모 | Recover Streak →` 버튼이 표시됩니다.
3. 버튼을 탭하면 확인 다이얼로그가 나타납니다.
4. 확인하면 어제 날짜가 completedDays에 추가되고 의지력 3이 차감됩니다.

### 복구가 표시되지 않는 경우

- 오늘 루틴을 이미 완료한 경우
- 챌린지 Day 1인 경우 (어제가 없음)
- 어제가 이미 완료된 경우
- 의지력이 3 미만인 경우
- 해당 챌린지에서 최근 7일 이내 이미 복구를 사용한 경우

### 설계 의도
레벨이 높을수록 의지력이 풍부해 복구 여유가 늘어납니다.  
오래 쌓아온 사용자일수록 "내 의지력이 나를 지켜준다"는 내러티브를 경험할 수 있습니다.

## 5. 의지력 & 레벨

### 의지력(Willpower)이란?
루틴을 완료할 때마다 **+1** 씩 쌓이는 누적 점수입니다.  
**절대 줄어들지 않으며**, 스트릭 복구 비용 차감 후에도 표시값이 감소하지 않습니다.

> 의지력 = 전체 완료 일수 합산 - 복구에 사용된 의지력

### 레벨 시스템

## 6. 배지 시스템

### 개요
다양한 조건을 달성하면 배지를 획득합니다. 총 **50개 배지**가 준비되어 있으며, 6개 카테고리로 분류됩니다.

기록 화면 상단의 **배지 컬렉션**에서 획득·미획득 배지를 모두 확인할 수 있습니다.

### 희귀도(Rarity)

| 희귀도 | 설명 |

| ⚪ Common | 달성하기 쉬운 기본 배지 |

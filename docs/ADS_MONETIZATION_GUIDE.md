# 게스트 모드 광고 수익화 가이드

> 비개발자를 위한 광고 수익화 완전 초보 가이드.  
> "앱에서 광고로 돈 버는 것"이 어떤 원리인지부터 실제 구현 방법까지.

---

## 1. 광고 수익의 기본 원리

### 돈이 오가는 흐름

```
광고주 (삼성, 나이키 등)
    ↓ 광고비 지불
광고 플랫폼 (Google AdMob)
    ↓ 수익 배분 (약 70%)
앱 개발자 (나)
```

앱에 직접 광고주를 구할 필요가 없습니다. **Google AdMob** 같은 광고 플랫폼이 중간에서 수천 개의 광고주 광고를 자동으로 배치해주고, 수익의 약 70%를 앱 개발자에게 줍니다.

### 수익 계산 방식

광고 수익은 주로 **CPM** 방식으로 계산됩니다.

> CPM = 광고 1,000번 노출당 받는 금액

예시:
- CPM $2 (한국 기준 평균)
- 하루 광고 노출 10,000번
- 하루 수익 = 10,000 / 1,000 × $2 = **$20 (약 2.8만원)**

## 2. 광고 종류 4가지

### 2-1. 배너 광고 (Banner Ad)
```
┌─────────────────────────┐
│      앱 콘텐츠           │
│                         │
│                         │
├─────────────────────────┤
│  [광고] 나이키 운동화 50%  │  ← 화면 하단에 항상 표시
└─────────────────────────┘
```
- **특징**: 화면 아래 고정, 사용자가 닫을 수 없음
- **수익**: 낮음 (CPM $0.5~2)
- **UX 영향**: 낮음 (거슬리지 않음)
- **Streakly 적용**: 홈 화면 하단에 항상 표시

---

### 2-2. 전면 광고 (Interstitial Ad)
```
┌─────────────────────────┐
│                         │
│   [광고 전체 화면]        │
│   삼성 갤럭시 S25        │
│                         │
│              [X] 닫기   │  ← 5초 후 닫기 가능
└─────────────────────────┘
```
- **특징**: 화면 전체를 덮음, 일정 시간 후 닫기 가능
- **수익**: 중간 (CPM $3~10)
- **UX 영향**: 높음 (갑자기 뜨면 불쾌함)
- **Streakly 적용**: 챌린지 완료 시 축하 화면 → 다음 화면 넘어갈 때

---

### 2-3. 보상형 광고 (Rewarded Ad)
```
┌─────────────────────────┐
│  광고 보고 보상 받기?     │
│                         │
│  ✅ 광고 보기            │  ← 유저가 자발적으로 선택
│  (스트릭 연장권 1개 받기) │
│                         │
│  ✖ 나중에              │
└─────────────────────────┘
```
- **특징**: 유저가 자발적으로 선택해서 봄, 보상 지급
- **수익**: 높음 (CPM $10~30)
- **UX 영향**: 없음 (강요 안 함)
- **Streakly 적용**: "오늘 스트릭 놓쳤을 때 광고 보고 복구하기"

---

### 2-4. 네이티브 광고 (Native Ad)
```
┌─────────────────────────┐
│  오늘의 루틴             │
│  ├ 운동하기  ✅          │
│  └ 물 마시기 ✅          │
├─────────────────────────┤
│  [광고] 추천             │  ← 앱 디자인과 비슷하게 섞임
│  Calm 명상 앱 - 지금 무료 │
└─────────────────────────┘
```
- **특징**: 앱 UI처럼 보이게 자연스럽게 배치
- **수익**: 중간
- **UX 영향**: 낮음 (자연스러움)
- **Streakly 적용**: 기록 화면 완료 챌린지 목록 사이

## 3. Streakly에서 광고를 어디에 넣을까

### 게스트 유저 (무료) vs 구독 유저 (PRO) 차이

| 위치 | 게스트 (광고 O) | PRO 구독자 (광고 X) |
|---|---|---|
| 홈 화면 하단 | 배너 광고 | 광고 없음 |
| 챌린지 완료 시 | 전면 광고 | 없음 |
| 스트릭 복구 시 | 보상형 광고 선택 가능 | 바로 복구 |
| 기록 화면 | 네이티브 광고 | 없음 |

### 광고 배치 원칙 (유저 이탈 방지)

**좋은 타이밍** (자연스러운 전환점):
- 챌린지 완료 직후
- 앱 재실행 후 첫 화면
- 새 챌린지 생성 완료 후

**나쁜 타이밍** (절대 하면 안 됨):
- 체크 버튼 누르는 중간
- 로그 작성 중
- 앱 처음 켰을 때 바로

## 4. 광고 플랫폼 선택

### 추천: Google AdMob

| 항목 | 내용 |
|---|---|
| 점유율 | 앱 광고 시장 1위 |
| 최소 지급액 | $100 (약 14만원) |
| 지급 주기 | 월 1회 |
| Flutter 지원 | ✅ 공식 플러그인 있음 |
| 계정 개설 | Google AdSense 계정으로 신청 |

### 보조 플랫폼 (나중에 추가 검토)
- **Meta Audience Network** — 페이스북/인스타 광고 네트워크
- **Unity Ads** — 게임 앱에 강하지만 일반 앱도 가능

## 5. 실제 수익 현실적 예측

Streakly 같은 습관 앱 기준:

| MAU | 하루 광고 노출 | 월 예상 수익 |
|---|---|---|
| 1,000명 | 약 3,000회 | $5~15 (7천~2만원) |
| 10,000명 | 약 30,000회 | $50~150 (7만~21만원) |
| 100,000명 | 약 300,000회 | $500~1,500 (70만~210만원) |

> **현실적 조언**: 광고 수익만으로 생계를 유지하려면 MAU 10만 명 이상이 필요합니다.  
> 초기에는 광고보다 **PRO 구독 전환 유도**가 더 중요합니다.

## 6. 구현 방법 (Claude Code로 작업 시)

### 필요한 작업 순서

**사람이 직접 해야 하는 것:**

1. **AdMob 계정 생성**
   - admob.google.com 접속
   - Google 계정으로 로그인
   - "앱 추가" → iOS/Android 각각 등록
   - **앱 ID** 복사 (나중에 필요)

2. **광고 단위 생성**
   - AdMob → 앱 선택 → 광고 단위 → 만들기
   - 배너 / 전면 / 보상형 각각 하나씩 만들기
   - **광고 단위 ID** 복사 (나중에 필요)

3. **앱 스토어 심사 주의사항**
   - iOS: 보상형 광고는 반드시 유저가 자발적으로 선택해야 함
   - 광고가 콘텐츠를 완전히 가리면 심사 반려될 수 있음

**Claude Code가 해주는 것:**

```
AdMob 연동해줘.
앱 ID는 ca-app-pub-xxxxxxxx~xxxxxxxx,
배너 광고 ID는 ca-app-pub-xxxxxxxx/xxxxxxxx야.
게스트 유저에게만 홈 화면 하단에 배너 광고 노출해줘.
PRO 구독자에게는 광고 안 보이게 해줘.
```

이렇게 전달하면 Claude Code가:
- `google_mobile_ads` 패키지 추가
- 배너 위젯 구현
- 게스트/PRO 분기 처리
- iOS/Android 설정 파일 수정

까지 전부 처리합니다.

## 7. 광고 vs 구독 전략 (중요)

광고와 구독은 **경쟁이 아니라 협력 관계**입니다.

```
게스트 유저가 광고를 보다가
    ↓
"광고 없애고 싶다" → PRO 구독 전환
```

광고는 수익원이기도 하지만, **PRO 구독의 가장 강력한 판매 도구**입니다.

### 전환을 높이는 UX 패턴

광고가 뜬 직후에:
```
┌─────────────────────────┐
│  광고 없이 사용하려면?    │
│                         │
│  [PRO로 업그레이드]      │  ← 이 타이밍에 전환율 높음
│  월 $2.99 · 광고 제거   │
└─────────────────────────┘
```

| 구현 | Claude Code에게 앱 ID 전달하면 됨 |
| 수익 기대치 | MAU 1만 명 기준 월 7~21만원 |
| 진짜 목표 | 광고 수익보다 PRO 구독 전환 유도 |

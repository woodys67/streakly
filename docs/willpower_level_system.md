# 의지력 레벨 시스템 설계

## 개요

- **의지력(Willpower)**: 모든 챌린지에서 루틴을 완료한 날의 총 합산 점수
- **획득 규칙**: 감점 없음 — 미완료일 0점, 완료일 +1점 (진행 중인 챌린지 수만큼)
- **총 레벨**: 25레벨
- **만렙 조건**: Lv.25 게이지 완성 = **누적 의지력 1,000점**

## 게이지바 계산 방식

```dart
// 레벨 진입 점수 테이블 (인덱스 0 = Lv.1 진입, 인덱스 25 = 만렙)
static const List<int> thresholds = [
    0, 10, 22, 37, 54,      // Lv.1–5  (🌱 씨앗)
   74, 96,121,148,178,      // Lv.6–10 (🌿 새싹)
  210,245,282,322,365,      // Lv.11–15(🌳 성장)
  410,458,508,561,616,      // Lv.16–20(🌸 결실)
  674,734,797,862,930,      // Lv.21–25(⚡ 전설)
  1000,                     // 만렙 완성
];

// 현재 레벨 (1~25)
int getLevel(int willpower) {
  final capped = willpower.clamp(0, 1000);
  for (int i = thresholds.length - 2; i >= 0; i--) {
    if (capped >= thresholds[i]) return i + 1;
  }
  return 1;
}

// 현재 레벨 내 게이지 진행률 (0.0 ~ 1.0)
double getLevelProgress(int willpower) {
  if (willpower >= 1000) return 1.0;
  final level = getLevel(willpower);
  final current = thresholds[level - 1];
  final next    = thresholds[level];
  return (willpower - current) / (next - current);
}
```

## 홈 화면 StreakCard 변경

### 제거
- D1~D7 도트 위젯 (`_DayDots`)

### 대체 (게이지바)
- **레벨 & 티어**: `Lv.N  🌱 씨앗` 등
- **게이지바**: 현재 레벨 내 진행률 (LinearProgressIndicator)
- **점수 표기**: `현재점수 / 다음레벨 진입점수`
